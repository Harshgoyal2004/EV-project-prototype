# EV Market Analysis and Prediction Prototype

An interactive Streamlit application for analyzing and predicting trends in the Indian Electric Vehicle (EV) market, with comprehensive market analysis, consumer segmentation, and predictive modeling.

## Overview

This application provides comprehensive analysis and visualization of the Indian EV market, including:

- Consumer segment profiles and characteristics
- Market analysis and trends
- Geographic distribution of EV adoption
- Financial projections and modeling
- Predictive analytics for future market development

## Features

The application includes 9 interactive sections accessible from the sidebar:

1. **Overview**: General information about the Indian EV market and key consumer segments
2. **Segment Profiles**: Detailed profiles of the four key consumer segments
3. **Market Analysis**: Analysis of vehicle categories, manufacturers, and sales trends
4. **Geographic Insights**: State-wise adoption and penetration rates
5. **Financial Projections**: Interactive financial modeling and projections
6. **Financial Equation**: Mathematical models for revenue calculation
7. **Business Model**: Business strategy and market approach
8. **Predictive Analytics**: Machine learning models for market prediction
9. **Market Forecast**: Future market trends and forecasts

## Data Sources

The application uses three main datasets:
- Electric vehicle sales by state
- Electric vehicle sales by manufacturer
- Consumer survey responses on EV adoption

## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/ev-market-analysis-prediction.git
cd ev-market-analysis-prediction

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run app.py
```

## Requirements

- Python 3.9+
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Plotly
- Scikit-learn
- Seaborn
